# THERUIZ AURA Prompt Builder

## 本地启动

```bash
npm install
npm run dev
```

在浏览器打开：

```text
http://localhost:5173
```

## 使用方式

1. 选择图片类型
2. 选择鞋款
3. 选择季节
4. 选择场景偏好，可选
5. 输入补充要求，可选
6. 点击生成提示词
7. 复制英文提示词
8. 在生图工具中上传鞋图并粘贴提示词

系统会根据季节、鞋款和图片类型自动匹配适合 THERUIZ AURA 风格的服装搭配方向。

系统默认使用 Compact Prompt 模式。后台仍保留完整的品牌规则、场景规则、穿搭库、鞋型保护、人物比例、动作、街景和负面限制逻辑，但最终只输出最关键的 8–12 句提示词，减少重复词和过长 prompt，提高团队复制效率和生图稳定性。

系统已加入敏感词弱化逻辑。后台会自动减少 sexy、seductive、revealing、sports bra、bodycon、beauty selfie 等容易引发生图偏色情或审核风险的词，并替换为 clean sleeveless top、active top、refined styling、real outfit check 等更安全、更符合 THERUIZ AURA 品牌调性的表达。

产品上脚图、对镜穿搭图和生活场景图会自动加入轻博主穿搭感，让画面更适合小红书和日常穿搭参考，但不会变成网红风。

产品上脚图、对镜穿搭图和生活场景图会自动加入增强版活人感，让人物更真实自然，但仍保持 THERUIZ AURA 的干净、体面和高级感。

产品静物图仍由后台自动判断画面风格，前台不增加额外配置项。

系统内置四季多套穿搭方案，每个季节至少 10 套。前台不需要选择穿搭，系统会根据鞋款、季节、图片类型和场景自动匹配，并在同一场景下提供更多搭配变化。

系统目标为“16 个核心场景 × 每场景 50 套 = 800 套后台穿搭库”。团队不需要手动选择，系统会根据鞋款、季节、图片类型和补充要求自动匹配，并在重复生成时自动轮换。

第一阶段已接入 4 个核心场景的场景级穿搭种子库：

- commute
- cafeExterior
- weekendCityWalk
- gymInterior

每个场景目前包含 10 套 seed outfits，用于验证自动选择、筛选、轮换和提示词拼接逻辑。后续将扩展为每个场景 50 套，最终形成 16 个核心场景 × 每场景 50 套 = 800 套后台穿搭库。

系统不会在前台显示这些穿搭选项，团队只需选择鞋款、季节、场景和图片类型，后台会自动匹配最合适的一套穿搭。

系统会根据鞋款、季节、场景和穿搭自动匹配适合的奢侈品风格包包或眼镜作为辅助搭配，但不会强调 Logo，也不会让配饰抢走鞋子和品牌氛围。

系统会根据不同场景自动匹配清晨、中午、傍晚的光线氛围。室内场景会自动避免墨镜，保持人物和场景真实自然。

系统会在后台自动匹配更多服装搭配风格，并表达 THERUIZ AURA 德训鞋“百搭、一鞋多场景、舒服但不随便”的产品特性，无需团队手动选择风格。

系统会在后台自动参考 Loro Piana、CHANEL、Chloé、Celine、The Row 等奢侈品风格的当季穿搭气质，但只参考材质、色彩、轮廓和高级感，不生成明显 Logo，不复制具体秀款，也不会让奢侈品元素抢走鞋子主体。

系统支持“健身房内”和“去运动的路上”两类运动生活方式场景，并内置 20 套轻运动穿搭方案。系统会自动匹配更适合 THERUIZ AURA 的运动前后生活方式内容，强调产品可用于轻运动、通勤与日常切换，但不会生成专业运动品牌式视觉。

系统会自动增加四季服装多样性，不再只生成浅色系穿搭。后台会根据鞋款、季节和场景自动加入真实城市女性衣柜中常见的深色锚点、牛仔、针织、外套、包袋和自然穿着细节，让画面更有活人感、更少 AI 模板感。

系统会自动锁定人物身份。所有人物图默认使用亚裔女生或亚裔混血女生；当生成两张或多张图片时，会自动强化同一模特、同一张脸、同一发型、同一妆容、同一身材比例和同一套穿搭的一致性，避免换脸和模特气质漂移。

夏季穿搭系统已加入裙装、短裤、T恤、背心、Polo、衬衫外搭、薄开衫等更多真实夏季单品。系统会根据不同场景和鞋款自动匹配颜色与款式，不需要团队手动选择。

健身房内场景已扩展支持深色服装配色、短裤、短袖、轻撸铁场景和高级健身房环境。系统会自动匹配更真实、更高级的健身房穿搭与空间氛围，同时保持 THERUIZ AURA 的轻运动生活方式调性。

系统会根据图片类型和场景自动判断模特视线。有些场景允许自然看镜头以增强互动感；对镜、静物、花絮、阅读、书店、花店等场景则会优先使用手机遮脸、任务型视线、侧看或低头，避免强摆拍和网红感。

系统会自动修复模特身材比例，避免 AI 常见的小头长腿、躯干过短、脚过小、镜拍拉腿、鞋腿关系错乱等问题。后台会根据站姿、走路、坐姿、镜拍、健身房等不同场景自动加强比例控制。

系统会根据场景自动匹配自然动作姿势，例如慢走、站立、拿咖啡、拿花、拿包、看书、调整衣服、健身房训练间隙等，并自动避免高风险复杂动作导致的手部错误、腿部变形、鞋子穿模和 AI 摆拍感。

系统会自动加强鞋型保护，避免上脚图、镜拍图、生活图、健身房图和静物图中出现鞋子穿模、鞋型变形、裤脚吃鞋、鞋底变厚、鞋头变宽、鞋带鞋舌消失、德训鞋变跑鞋等问题。

当鞋子需要完整露出时，系统会自动加强鞋带自然打结状态，避免鞋带未系、融化、飞散、线团化、鞋带结不可读或双鞋鞋带状态不一致等问题。

系统已加入场景真实感补丁。后台会根据街拍、店铺、酒店、超市、健身房、书店、画廊、静物、拍摄花絮等不同场景自动强化真实空间逻辑、真实光线、真实生活细节和低噪音背景，避免 AI 布景感和假高级感。
