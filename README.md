# THERUIZ AURA Prompt Builder

## 本地启动

```bash
npm install
npm run dev
```

在浏览器打开：

```text
http://localhost:5173
```

## 使用方式

1. 选择图片类型
2. 选择鞋款
3. 选择季节
4. 选择场景偏好，可选
5. 选择服装类型，可选
6. 输入补充要求，可选
7. 点击生成提示词
8. 复制英文提示词
9. 在生图工具中上传鞋图并粘贴提示词

系统会根据季节、鞋款和图片类型自动匹配适合 THERUIZ AURA 风格的服装搭配方向。

系统已加入“10 核心场景服装种子库 + Outfit Intelligence v2 + Stability Guard 穿搭智能选择系统”。后台建立了 10 个核心场景 × 每场景 12 套，共 120 套基础穿搭种子。系统不再简单随机选择穿搭，而是根据场景、鞋款、季节、图片类型、服装类型、城市气质、用户补充要求和历史生成记录，对候选穿搭进行硬过滤、评分、风险惩罚和去重轮换。

系统新增稳定性保护机制，包括种子库质量验收、最低通过分 65、用户补充要求优先级、前台服装类型与补充要求冲突处理、fallback 安全穿搭模板、localStorage 生成历史、穿着段 prompt 预算控制。该系统会提升穿搭丰富性、鞋款适配度、颜色变化、活人感和轻网感，同时降低同场景重复、遮鞋、手持物冲突、敏感词、AI 模板化和提示词过长风险。

系统已加入“四季 × 6 城市时间 / 光线 / 穿搭厚度自动匹配总补丁”。后台会根据季节、城市、场景、图片类型和鞋款，自动判断时间段、光线、照片风格、空气感、穿搭厚度、秋冬材质和负面限制。不同季节不再只是换服装，而是会自动区分照片光线、色彩、阴影、材质表现和画面情绪：春季更干净清新，夏季更明亮透气，秋季更有温暖材质层次，冬季更克制安静但不阴暗。

系统已建立四季 × 上海、成都、深圳、杭州、北京、泛中国城市的自动匹配矩阵。北京冬季会更偏冷干和完整保暖层次；上海/杭州偏湿冷和轻羊毛叠穿；成都偏温和松弛叠穿；深圳偏轻秋冬和现代清爽层次；泛中国城市使用中等厚度。系统会避免南方城市穿得过厚、北方冬季穿得过薄、夏季暴晒、春季过甜、秋季橙黄滤镜和冬季死灰阴暗。

系统已加入“提示词稳定性总控 + 室内外光线区分总补丁”。系统会将 prompt 内容分为硬约束和软审美，确保鞋型、鞋子完整可见、人物比例、室内外光线、单一主手持物、敏感词安全等关键内容永远优先保留。后台会在生成前进行 preflight 检查，自动修复多个主手持物、室内外光线错位、每张图默认有包、鞋子被挡、季节穿搭冲突、opening / pants / lower body 词汇残留、多相机风格冲突等问题。

系统同时建立图片类型专属 prompt 骨架，包括人物街拍、对镜穿搭、室内商业空间、健身房、产品静物、产品细节图。不同图片类型会自动保留不同重点，减少一个大模板控制所有场景导致的不稳定。

系统已加入鞋款专属保护层。Cloud Dancer、Sand Dollar、Cappuccino、Silver Romance、Aire、Delphinium Blue、Lemon、Maple Grove、Oreo、Panda 会分别匹配专属材质、颜色和变形风险控制。

系统已加入室内 / 室外光线区分逻辑，会根据场景自动判断户外街景、室内自然光、室内商业空间、健身房室内、产品静物、半室内门口光线，避免室内出现户外街拍光、户外出现棚拍窗光、健身房变运动广告光、静物图变 CGI 渲染光、门口场景出现室内外光线冲突。

系统会自动替换容易造成歧义或 body-focused 风险的表达。最终 prompt 不输出 opening / pants / lower body。鞋口相关表达统一替换为 sneaker collar / ankle entry area；裤装相关表达统一替换为 trousers / denim / shorts / active trousers；人物比例相关表达统一替换为 full figure balance / leg-and-sneaker proportion / stance and foot placement 等更安全准确的表达。

系统已升级为 Standard Accurate Prompt 模式。默认输出中长提示词，优先保证场景、城市街景、人物动作、穿搭、真实相机质感和鞋型保护的准确性。

系统已升级为结构化提示词呈现方式。最终 prompt 将按照“时间 / 地点 / 人物 / 穿着 / 场景 / 氛围 / 动作 / 负面词”的结构输出，便于团队检查和复制使用。该结构不会削弱原有的鞋型保护、城市街景、真实相机质感、穿搭多样性、场景真实化和敏感词弱化逻辑，只是将最终提示词变成更清晰的拍摄 brief。

静物图会自动省略人物和动作，并使用“产品”字段突出鞋型、材质、鞋带、鞋舌、鞋底和产品比例。

系统已加入场景穿搭多样性与色彩活人感逻辑。前台仅新增“服装类型”选项，可选择自动匹配、裤装、裙装、短裤、连衣裙、轻运动。后台会根据场景、季节、鞋款和生成历史自动轮换服装类型、款式、颜色方向和轻网感细节，避免同一场景人物穿搭模板化。

系统已加入中国城市真实街景与 6 城市场景自动匹配逻辑。户外人物图会根据场景自动匹配上海、成都、深圳、杭州、北京或泛中国城市的日常街区气质，避免欧美街拍、地标旅游、乱码招牌和城市宣传片式背景。

系统已加入真实相机拍摄质感补丁。后台会根据图片类型和场景自动选择 Leica-inspired 真实街拍感、Hasselblad-inspired 高级静物/空间感或 Fujifilm-inspired 柔和日常色彩感，降低 AI 渲染感、塑料感和过度修图感。

系统继续保留敏感词弱化逻辑，避免性感、擦边、body-focused、运动内衣广告感等方向，保持 THERUIZ AURA 的轻熟、体面、真实和高级。

系统已加入敏感词弱化逻辑。后台会自动减少 sexy、seductive、revealing、sports bra、bodycon、beauty selfie 等容易引发生图偏色情或审核风险的词，并替换为 clean sleeveless top、active top、refined styling、real outfit check 等更安全、更符合 THERUIZ AURA 品牌调性的表达。

产品上脚图、对镜穿搭图和生活场景图会自动加入轻博主穿搭感，让画面更适合小红书和日常穿搭参考，但不会变成网红风。

产品上脚图、对镜穿搭图和生活场景图会自动加入增强版活人感，让人物更真实自然，但仍保持 THERUIZ AURA 的干净、体面和高级感。

产品静物图仍由后台自动判断画面风格，前台不增加额外配置项。

系统内置四季多套穿搭方案，每个季节至少 10 套。前台不需要选择穿搭，系统会根据鞋款、季节、图片类型和场景自动匹配，并在同一场景下提供更多搭配变化。

系统目标为“16 个核心场景 × 每场景 50 套 = 800 套后台穿搭库”。团队不需要手动选择，系统会根据鞋款、季节、图片类型和补充要求自动匹配，并在重复生成时自动轮换。

第一阶段已接入 4 个核心场景的场景级穿搭种子库：

- commute
- cafeExterior
- weekendCityWalk
- gymInterior

每个场景目前包含 10 套 seed outfits，用于验证自动选择、筛选、轮换和提示词拼接逻辑。后续将扩展为每个场景 50 套，最终形成 16 个核心场景 × 每场景 50 套 = 800 套后台穿搭库。

系统不会在前台显示这些穿搭选项，团队只需选择鞋款、季节、场景和图片类型，后台会自动匹配最合适的一套穿搭。

系统已加入“场景配饰智能选择补丁”。配饰不再默认每张图都有包，而是根据场景、季节、城市、服装、动作、图片类型和主手持物自动判断。系统会在 noAccessory、wearableOnly、bagAsSecondaryAccessory、bagAsPrimaryHandheldObject、handheldOnly、minimalMixed 等策略中选择最合适的一种。包袋只在支持场景逻辑时出现，不再作为默认穿搭项。

系统会严格控制每张人物图最多一个主手持物和最多两个轻配饰。对镜图手机就是主手持物；健身房内不默认有包；花店有花束时通常不再加包；咖啡店有咖啡时包只能作为肩背配件或取消；精品超市和通勤可有包，但不会每次都有。配饰不得遮挡鞋子、不得穿模、不得抢 THERUIZ AURA 鞋子的主体关系。

系统会根据不同场景自动匹配清晨、中午、傍晚的光线氛围。室内场景会自动避免墨镜，保持人物和场景真实自然。

系统会在后台自动匹配更多服装搭配风格，并表达 THERUIZ AURA 德训鞋“百搭、一鞋多场景、舒服但不随便”的产品特性，无需团队手动选择风格。

系统会在后台自动参考 Loro Piana、CHANEL、Chloé、Celine、The Row 等奢侈品风格的当季穿搭气质，但只参考材质、色彩、轮廓和高级感，不生成明显 Logo，不复制具体秀款，也不会让奢侈品元素抢走鞋子主体。

系统支持“健身房内”和“去运动的路上”两类运动生活方式场景，并内置 20 套轻运动穿搭方案。系统会自动匹配更适合 THERUIZ AURA 的运动前后生活方式内容，强调产品可用于轻运动、通勤与日常切换，但不会生成专业运动品牌式视觉。

系统会自动增加四季服装多样性，不再只生成浅色系穿搭。后台会根据鞋款、季节和场景自动加入真实城市女性衣柜中常见的深色锚点、牛仔、针织、外套、包袋和自然穿着细节，让画面更有活人感、更少 AI 模板感。

系统会自动锁定人物身份。所有人物图默认使用亚裔女生或亚裔混血女生；当生成两张或多张图片时，会自动强化同一模特、同一张脸、同一发型、同一妆容、同一身材比例和同一套穿搭的一致性，避免换脸和模特气质漂移。

夏季穿搭系统已加入裙装、短裤、T恤、背心、Polo、衬衫外搭、薄开衫等更多真实夏季单品。系统会根据不同场景和鞋款自动匹配颜色与款式，不需要团队手动选择。

健身房内场景已扩展支持深色服装配色、短裤、短袖、轻撸铁场景和高级健身房环境。系统会自动匹配更真实、更高级的健身房穿搭与空间氛围，同时保持 THERUIZ AURA 的轻运动生活方式调性。

系统会根据图片类型和场景自动判断模特视线。有些场景允许自然看镜头以增强互动感；对镜、静物、花絮、阅读、书店、花店等场景则会优先使用手机遮脸、任务型视线、侧看或低头，避免强摆拍和网红感。

系统会自动修复模特身材比例，避免 AI 常见的小头长腿、躯干过短、脚过小、镜拍拉腿、鞋腿关系错乱等问题。后台会根据站姿、走路、坐姿、镜拍、健身房等不同场景自动加强比例控制。

系统会根据场景自动匹配自然动作姿势，例如慢走、站立、拿咖啡、拿花、拿包、看书、调整衣服、健身房训练间隙等，并自动避免高风险复杂动作导致的手部错误、腿部变形、鞋子穿模和 AI 摆拍感。

系统会自动加强鞋型保护，避免上脚图、镜拍图、生活图、健身房图和静物图中出现鞋子穿模、鞋型变形、裤脚吃鞋、鞋底变厚、鞋头变宽、鞋带鞋舌消失、德训鞋变跑鞋等问题。

系统已加入“活人感 + 人物身材比例 + 鞋子上脚比例稳定总补丁”。后台会根据图片类型、场景、动作、服装类型和鞋款，自动增强人物真实状态、穿搭穿在身上的自然感、真实手部动作、身体重心、头身比、肩颈、腰线、腿长、手脚比例，以及脚踝、鞋口、小腿、裤脚、裙摆、袜子、鞋底接地、鞋头朝向和鞋脚比例。该补丁用于降低 AI 假人感、身体比例漂移、镜拍拉腿、鞋子像贴图、鞋口塌陷、裤脚吃鞋、鞋底漂浮和德训鞋上脚不真实的问题。

系统已加入“单一主手持物 + 手持物防穿模总补丁”。后台会根据场景和补充要求自动选择唯一主手持物，避免咖啡、花、书、纸袋、水瓶、手机、手提包等多个主物体同时出现在手上，并自动加强手指、手掌、手腕、袖子、包带、身体、腿部、鞋子和物体边缘之间的空间分离，降低手物融合、漂浮道具、包袋挡鞋和手持物穿模问题。

当鞋子需要完整露出时，系统会自动加强鞋带自然打结状态，避免鞋带未系、融化、飞散、线团化、鞋带结不可读或双鞋鞋带状态不一致等问题。

系统已加入场景真实感补丁。后台会根据街拍、店铺、酒店、超市、健身房、书店、画廊、静物、拍摄花絮等不同场景自动强化真实空间逻辑、真实光线、真实生活细节和低噪音背景，避免 AI 布景感和假高级感。
