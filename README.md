# THERUIZ AURA Prompt Builder

THERUIZ AURA Prompt Builder 用于生成结构化生图提示词，帮助团队稳定输出符合 THERUIZ AURA 视觉系统的本地 prompt。

## 前台字段

- 图片类型
- 鞋款
- 季节
- 场景偏好
- 服装类型
- 补充要求

## v1.0 核心能力

- 结构化 prompt
- 鞋型保护
- 人物比例稳定
- 鞋子上脚比例稳定
- 中国城市 / 四季 / 室内外光线
- 简化版穿搭选择
- 服装类目多样性
- 配饰智能选择
- 单一主手持物
- 相机质感自动切换
- 敏感词与词汇替换
- finalPromptSafetyCheck

## 本地使用

```bash
npm install
npm run dev
npm run build
```

本地打开：

```text
http://localhost:5173
```

## 棚内上新拍摄

- 支持五种棚拍背景预设：暖灰无影棚、奶油白极简棚、亚麻质感背景棚、黑白灰结构棚、自然窗光模拟棚。
- 支持 20 件棚拍服装单品，按服装类型和季节筛选。
- 自动模式根据季节、鞋款、服装类型和背景自动匹配单品。
- 用户可明确选择棚拍服装单品，选择后不会被随机替换。
- 同一 8 张系列保持同一服装、同一棚景、同一灯光方向。
- 所有服装服务于 THERUIZ AURA 德训鞋人物图，严格保持鞋型保护、鞋履可见性和防穿插规则。
- 当前只生成 Prompt，图片生成接口仍未接入。
- 当前不代表已支持服装产品白底图或服装 Product Truth。

## 注意事项

- 不默认每张图有包。
- 每张人物图最多一个主手持物。
- 静物图无人物。
- 对镜图手机为主手持物。
- 最终 prompt 不输出 opening / pants / lower body。
- 每次只输出一种相机质感。
- v1.0 不使用复杂评分系统。
